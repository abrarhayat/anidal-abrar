## Build and run
### To Run Locally:
Clone the repository and perform the following command line actions:

```bash
$> cd anidal
$> npm install
$> npm run start
```

### To build/run docker container:

```bash
$> docker build -t game .
$> docker run -d -p 3000:3000 game
```

Open [http://localhost:3000](http://localhost:3000) in browser.
